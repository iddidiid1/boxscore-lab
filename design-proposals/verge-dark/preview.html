Design Token Comparison
Token / Property Current Proposed Notes
Canvas background #0b1326 #131313 Warmer near-black vs cool navy. Less blue cast overall.
Card / panel surface #121c2f #1a1a1a Follows the canvas shift — neutral dark vs tinted dark.
Primary action color #2563eb (blue) #3cffd0 (mint) Biggest color change. All CTAs, active states, focus rings shift to mint.
Secondary accent — #5200ff (ultraviolet) New: used for alternative tags, badge variants, promotional elements.
Success / positive color #10b981 (emerald) #3cffd0 (mint, same as primary) Unifies success with the primary accent. Tradeoff: less differentiation.
Button border radius 4px 24px (pill) Most visible shape change. Full pill vs near-square.
Card border radius 6px 16px Softer, more modern. Maintains information density.
Badge border radius 2px 20px (pill) Follows card direction — rounder, less typewriter-stamp feel.
Input border radius 4px 4px (unchanged) Keeping forms tight/functional is intentional for data entry.
Display / hero font Hanken Grotesk Anton (Manuka substitute) Used for page titles, team names in detail views. More editorial presence.
Body / UI font Inter Space Grotesk Subtle shift — similar legibility, slightly wider letter forms. Could keep Inter.
Data / stats font JetBrains Mono JetBrains Mono (unchanged) Already correct. No change.
Label style Mixed case, body font ALL CAPS, JetBrains Mono, 1.5–2px tracking Section headers, table headers, form labels, eyebrows all shift.
Card border color #334155 (solid slate) rgba(255,255,255,0.18) Translucent white instead of tinted slate. Adapts better to any canvas color.
Elevation / depth Tonal layering (navy shades) 1px borders + mint accent on hover Same flat philosophy, different execution. Mint hover border replaces blue.
